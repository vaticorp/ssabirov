Страницы и запросы приложения:

  1. /start - стартовая страница приложения
  2. /login - страница регистрации и авторизация пользователя
  3. /register - страница преобразования ссылок и перехода
  4. /statistic - страница для просмотра актуальной статистики
  5. /help - вспомогательная страница
  6. /account - запрос для регистрации нового пользователя

Дополнительная информация


Для реализации основной логики использован стек технологий: Spring-boot, Hibernate, jQuery. Для приложения используется встроенная база данных h2. Для отображения страниц используется связка html, css, thymeleaf(как шаблонизатор)
